A simple text give a beer widget.

== Description ==

Built with : http://admin-builder.com

You can activate a widget that allows the users to give you a beer, virtually (appreciate your site, post, page, product).
A user with a IP will only be able to give you a beer once


== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the plugin files to the `/wp-content/plugins/give-a-beer-one` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Use the give a beer One page screen to configure the plugin (import the data generated by the service. More info in that page)
